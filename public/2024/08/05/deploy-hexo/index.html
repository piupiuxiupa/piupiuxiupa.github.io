3. 配置 GitHub

  1. 新建一个github项目

  2. 点进项目找到Settings -> Pages 进行配置 GitHub Pages

配置SSH方式

GitHub从2021年8月13号开始不能用密码推送，需要用token。

token需要在右上角个人Settings -> Developer Settings -> Personal access tokens里生成。

注意保管，丢失了需要重新生成。
